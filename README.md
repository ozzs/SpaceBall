## Instructions

**Move with A and D** ⬅️ ➡️

**Jump with spacebar**

**Avoid all obstacles as much as you can**

## Features
* Randomly Generated Obstacles
* Dynamic Wave System by Score
* Multiple Obstacle Supported by Prefabs
* WebGL
* Animations
* Post-Processing
* Dynamic Emission
* Particle System
* High Score System
* Menus UI: Main, Death & Pause
* Music & Sliders
